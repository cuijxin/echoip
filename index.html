What is my IP address?

{{ .IP }}

Multiple command line HTTP clients are supported, including curl, GNU Wget and fetch.

CLI examples:

$ curl ifconfig.co
{{ .IP }}

$ wget -qO- ifconfig.co
{{ .IP }}

$ fetch -qo- http://ifconfig.co
{{ .IP }}

Country lookup:

$ http ifconfig.co/country
{{ .Country }}

JSON output:

$ http --json ifconfig.co
HTTP/1.1 200 OK
Content-Length: 61
Content-Type: application/json
Date: Fri, 15 Apr 2016 17:26:53 GMT

{
    "country": "{{ .Country }}",
    "ip": "{{ .IP }}"
}

To force IPv4 lookup you can use the v4 subdomain (or v6 for IPv6).
