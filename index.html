What is my IP address?

{{ .IP }}

Multiple command line HTTP clients are supported, including curl, httpie, GNU Wget and fetch.

CLI examples:

$ curl ifconfig.co
{{ .IP }}

$ http -b ifconfig.co
{{ .IP }}

$ wget -qO- ifconfig.co
{{ .IP }}

$ fetch -qo- http://ifconfig.co
{{ .IP }}
{{ if .CountryLookup }}

Country lookup:

$ http ifconfig.co/country
{{ .Country }}
{{ end }}

Content-Type: application/json
